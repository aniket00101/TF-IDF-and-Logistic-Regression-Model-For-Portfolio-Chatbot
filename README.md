# 🤖 Aniket's Portfolio Chatbot — Backend

Intent-based chatbot using **TF-IDF + Logistic Regression**, served via **FastAPI**.

---

## 📁 Project Structure

```
chatbot-backend/
│
├── dataset/
│   └── training_data.csv        ← Labeled training sentences (text, intent)
│
├── model/
│   └── train_chatbot.ipynb      ← Jupyter notebook: preprocess → train → evaluate → save
│
├── trainedmodel/
│   └── intent_model.pkl         ← Saved Pipeline (TF-IDF + LR) — generated by notebook
│
├── main.py                      ← FastAPI server (loads .pkl, serves /chat endpoint)
├── Chatbot.jsx                  ← React frontend component (calls FastAPI via fetch)
├── requirements.txt             ← All Python dependencies
└── README.md
```

---

## 🚀 Getting Started

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Open the Jupyter notebook and train the model
```bash
jupyter notebook model/train_chatbot.ipynb
```
Run all cells → this saves `trainedmodel/intent_model.pkl`

### 3. Start the FastAPI server
```bash
uvicorn main:app --reload --port 8000
```

### 4. Test the API
```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "what skills do you have"}'
```

Expected response:
```json
{
  "reply": "Here's Aniket's full tech stack:\n\n🎨 Frontend : ...",
  "intent": "skills",
  "confidence": 0.91
}
```

## ☁️ Deploy to Render (Free)

1. Push this folder to a **new GitHub repo**
2. Go to [render.com](https://render.com) → **New Web Service** → connect repo
3. Set these:
   - **Build command:** `pip install -r requirements.txt && python -c "import nltk; nltk.download('stopwords'); nltk.download('wordnet'); nltk.download('omw-1.4')"`
   - **Start command:** `uvicorn main:app --host 0.0.0.0 --port $PORT`
4. Copy your Render URL (e.g. `https://aniket-chatbot.onrender.com`)
5. In your React project, set the env variable:
   ```
   VITE_CHATBOT_API_URL=https://aniket-chatbot.onrender.com
   ```

> ⚠️ You must commit `trainedmodel/intent_model.pkl` to GitHub before deploying, since Render won't run the notebook.

## 🧠 How It Works

```
User message
     ↓
 preprocess()        lowercase, remove punctuation, lemmatize, remove stopwords
     ↓
 TF-IDF Vectorizer   convert text → numeric feature vector (unigrams + bigrams)
     ↓
 Logistic Regression predict intent + confidence score
     ↓
 confidence < 0.35?  → fallback response
     ↓
 build_response()    return portfolio-specific reply
     ↓
 JSON response       { reply, intent, confidence }
```

## ➕ Adding New Intents

1. Add new rows to `dataset/training_data.csv`
2. Re-run `model/train_chatbot.ipynb` (all cells)
3. Add response logic in `main.py` → `build_response()`
4. Restart the server
